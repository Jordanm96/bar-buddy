## Overview
Bar Buddy is an app for all consumers of alcoholic beverages. This app will allow you to search through a database of over 500 drinks and cocktails to find the right dirnk you are looking for.
For bartenders: Use the search bar to find that unusual drink a customer requested, and recieve a list of all the ingredients and instructions for preparing the cocktail.
For casual users: If you are unsure of what you want to drink, shake it up! Use the random drink generator to make that beverage decision for you and display a random cocktail for you to make.

## Technical
Using a free third-party API from thecocktaildb.com, I used Vanilla Javascript and jQuery to render searched and random cocktails.
